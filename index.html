About ITMD 541

The ITMD 541 course's goal is to examine core web technologies that are integral in the creation of web-based applications typically delivered in a browser.

During the course we will cover:

  • fundamental web protocols
  • web application architectures
  • markup
  • scripting languages

We will also focus on writing modern, standards-compliant JavaScript and how it is used to interact with HTML and CSS to enable rich user interfaces and communication with other services. Current frameworks, libraries, and tools will also be explored.
